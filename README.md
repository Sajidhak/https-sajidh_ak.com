# Personal Portfolio Website

This is a simple, responsive personal website created using HTML, CSS, and JavaScript.  
The website presents my academic background, interests, and experience in a clean and professional manner.

## Purpose
The purpose of this website is to provide a formal online presence that highlights my interest in Physics and STEM education.  
The design focuses on clarity, readability, and accessibility without using images.

## Features
- Single-page responsive layout
- Clean and professional design
- Smooth scrolling and subtle animations
- Mobile-friendly
- No external frameworks used

## Technologies Used
- HTML5
- CSS3
- JavaScript (for scroll-based animations)
